# Element Breakthrough Game

中文名：元素觉醒录

这是一个闯关动作 RPG / Roguelite 游戏想法记录文档。后续新的玩法、角色、怪物、元素、关卡和系统设计，都可以继续追加到这里。

## 核心概念

玩家从第一关开始不断闯关。每一关都有普通怪物和关底 Boss。怪物身上带有一个或多个“元素”，这些元素代表怪物的能力、状态效果、行动方式或特殊属性。

击杀怪物后，怪物身上的元素会随机掉落一个或多个。玩家拾取后，会获得对应的元素能力；如果其他怪物拾取，也会获得同样的能力。

击败本关 Boss 后，玩家可以从 Boss 身上获得一种元素，并将该元素永久装配。Boss 还有一定几率成为玩家的宠物或坐骑。

## 主角职业

- 剑师：近战职业，适合正面战斗和高风险贴身输出。
- 射手：远程物理职业，适合保持距离、风筝和精准打击。
- 法师：远程法术职业，适合使用元素能力、范围伤害和控制效果。

## 元素机制

元素可以来自怪物、Boss 或关卡奖励。元素可能提供以下效果：

- 攻击能力
- 防御能力
- 移动方式
- 控制效果
- 持续伤害
- 召唤能力
- 宠物或坐骑相关能力

示例怪物：

- 蛇妖
  - 毒气重：带有眩晕或控制状态。
  - 蠕动：行动缓慢。

## 掉落与拾取

- 怪物死亡时，会随机掉落一个或多个自身元素。
- 玩家拾取元素后，会临时获得该元素能力。
- 其他怪物也可以拾取元素，并获得对应强化。
- 元素掉落会让战场持续变化，玩家需要判断先抢元素、先杀怪，还是控制怪物不要拿到强力元素。

## Boss 奖励

击败每关 Boss 后：

- 玩家可以获得一种 Boss 元素。
- 该元素会成为永久装配能力。
- Boss 有一定概率成为玩家宠物或坐骑。

## 第一关场景：青藤试炼林

青藤试炼林是第一关的入口区域。这里看起来像一片安静的森林训练场，但藤蔓深处藏着被元素污染的怪物。

画面结构：

- 左侧：主角出生点，一块带草皮的木质平台。
- 中段：低矮草坡和一块写着“试炼开始”的木牌。
- 右侧：怪物巡逻区域，地面有绿色毒雾从裂缝中冒出。
- 背景：蓝天、白云、远处树冠、藤蔓缠绕的小木屋。

关卡目标：

- 击败第一个怪物。
- 观察怪物身上的元素。
- 怪物死亡后掉落元素，玩家可以拾取。

可出现元素：

- 青藤：让攻击附带短暂缠绕效果。
- 毒雾：制造小范围毒气，可能造成眩晕或持续伤害。
- 蠕动：移动速度降低，但受到击退效果减少。

首次对白：

- 主角：“这里就是第一关？看起来不像很危险。”
- 怪物：“嘶……把你的元素留下！”
- 提示文字：“击败怪物后，抢先拾取掉落元素。”

## 第一版主角：赤羽剑师

职业：剑师

定位：近战输出，适合第一关教学。

页面表现：

- Q 版比例，红色高马尾，轻甲，手持短剑。
- 站立时剑尖朝下，进入战斗后身体前倾。
- 攻击时出现橙黄色半月剑光。
- 受击时有夸张后仰动作，方便玩家读懂反馈。

基础能力：

- 普通攻击：向前挥剑，造成近距离伤害。
- 冲刺斩：短距离突进并挥出一道剑光。
- 元素拾取：触碰掉落元素后，临时获得对应能力。

初始元素：

- 无。第一关通过击败怪物学习元素拾取。

性格关键词：

- 勇敢
- 有点急性子
- 喜欢正面突破

## 第一版怪物：毒藤蛇妖

类型：普通怪物

定位：第一关教学怪，用来展示“怪物自带元素”和“死亡掉落元素”。

页面表现：

- Q 版蛇妖，圆头、大眼、身体短粗。
- 身上缠着绿色藤蔓，移动时贴地蠕动。
- 嘴边有淡绿色毒气。
- 被攻击时弹起，死亡时元素从身体里飞出。

携带元素：

- 毒雾：靠近目标时释放小范围毒气，可能造成眩晕或持续伤害。
- 蠕动：行动缓慢，但不容易被击退。

行为规则：

- 平时在右侧平台慢速巡逻。
- 玩家靠近后，先停顿，再喷出一小团毒雾。
- 生命值降低后，会尝试靠近掉落元素并拾取。

死亡掉落：

- 有较高概率掉落“毒雾”。
- 有较低概率掉落“蠕动”。
- 玩家或其他怪物都可以拾取掉落元素。

### 优质场景建模与美术方向

#### 1. Q 版横版森林入口

适合第一关“青藤试炼林”。

- 前景：草叶、蘑菇、碎石、木桩，用来遮挡地面边缘。
- 中景：可行走平台、木牌、藤蔓、怪物巡逻区域。
- 远景：蓝天、白云、树冠、小木屋、山体剪影。
- 动态元素：飘动树叶、轻微摇晃的藤蔓、地面毒雾。
- 玩法结合：毒藤蛇妖死亡后，元素球从身体弹出并落在平台上。

#### 2. 毒雾沼泽

适合第二或第三关，用来扩展毒系元素。

- 场景关键词：绿色雾气、浅水、腐木桥、冒泡泥潭、发光菌类。
- 怪物设计：携带毒雾、迟缓、隐身、吸血等元素。
- 玩法结合：毒雾会停留在低洼地形；风元素可以吹散毒雾；火元素点燃毒气会造成爆炸。

#### 3. 熔岩矿洞

适合火系与岩石系 Boss。

- 场景关键词：红橙色岩浆、黑色岩壁、矿车轨道、断裂平台、火星粒子。
- 怪物设计：火蜥蜴、熔岩傀儡、爆裂矿虫。
- 玩法结合：火元素提升伤害但可能点燃场景；冰元素可以临时冷却岩浆生成落脚点。

#### 4. 雷云高塔

适合雷系、风系和飞行怪物。

- 场景关键词：高空平台、木塔、乌云、闪电、风车、悬空铁链。
- 怪物设计：雷鸟、风灵、磁甲守卫。
- 玩法结合：金属平台会导电；风元素影响跳跃距离；雷电可以激活机关。

#### 5. Boss 房间

每关 Boss 房间需要比普通场景更有仪式感。

- 入口区域要收窄，让玩家意识到即将进入战斗。
- Boss 出场时，背景元素先动起来，例如藤蔓收缩、毒雾聚合、火山震动。
- Boss 身上的元素要可视化，例如背后漂浮元素符号、身体颜色变化、武器附带特效。
- 击败后，永久元素以更大的光球出现，和普通掉落区分开。

### 本项目优先吸收的方向

第一阶段只吸收三件事：

- 横版 Q 版森林平台作为第一关视觉基调。
- 元素掉落、拾取、怪物也能拾取，作为核心玩法。
- Boss 永久元素和宠物/坐骑概率，作为关卡完成奖励。

暂不加入复杂大地图、多人联机、无限装备词条和大型剧情系统，避免早期设计过重。

## MVP-001：网页应用版本

- 技术栈：Vue + Vite。
- 游戏范围：第一关，包含一个主角、5-8 个不同普通怪物和第一关 Boss。

- 标记方式：直接记录在 `AGENTS.md` 和 `README.md`，不创建单独标记文件。

### MVP-001 运行方式

- 安装依赖：`npm.cmd install`
- 本地开发：`npm.cmd run dev -- --port 5173`
- 生产构建：`npm.cmd run build`
- 访问地址：`http://127.0.0.1:5173`

- Vue + Vite 网页应用。
- Q 版横版森林场景“青藤试炼林”。
- 键盘移动与攻击：方向键 / A D 移动，J 攻击，R 重新开始。
