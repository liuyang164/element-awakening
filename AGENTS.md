# Codex Project Instructions

This repository is for an action RPG / roguelite level-clear game prototype.

Before making design or implementation changes, read `README.md` for the current game idea.

Keep changes simple and focused:

- Put game design notes, new ideas, mechanics, monsters, elements, and TODOs in `README.md`.
- Preserve existing `README.md` content by default. Append new ideas or make only small targeted edits unless the user explicitly asks to rewrite or remove old content.
- Only add implementation details to this file when they affect how Codex should work in this repository.
- Do not overbuild systems before the gameplay requirement is clear.
